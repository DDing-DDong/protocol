## 🎮 기본 게임 감각 정의

### 해커 조작감 (팀원 A가 조정)

| 요소 | 현재값 | 목표 | 이유 |
|------|-------|------|------|
| 이동 속도 | 250px/s | 자연스러운 걷기 | 너무 빠르면 조작 어려움 |
| 점프 높이 | 620 초기속도 | 약 250px 상승 | 2단 플랫폼 높이 기준 |
| 대시 거리 | 620px/s × 0.85s | 약 530px 전진 | 화면 절반 거리 |
| 대시 쿨타임 | 0.85초 | 가능한 빈번하게 | 에너지 소모가 주 제약 |
| 에너지 소모 | 18 (대시), 24 (방어막/초) | 전략적 선택 필요 | 무제한 사용 방지 |
| 에너지 회복 | 10/초 | 약 10초 충전 | 충동적 플레이 안 하도록 |

**조정 기준:**
- 조작감이 "부드럽고 반응적"이어야 함
- 플레이어가 "내 실력이 부족"이라고 느껴야 함 (그냥 게임이 어려운 게 아니라)
- 반복 플레이할 수록 느는 게 있어야 함

### 함정 효과와 밸런스 (팀원 C가 조정)

#### 공격 턴 기본 함정

| 함정 | 타입 | 효과 | 높이 | 난이도 |
|------|------|------|------|--------|
| 레이저 | 수직 | -1 체력, 탐지 | 118px | 낮음 |
| 감전 바닥 | 수평 | 지연 1초 | 14px | 낮음 |
| 카메라 | 범위 | 탐지만 | 70px | 낮음 |

**특징:**
- 레이저: 시각적으로 명확 → 피하기 쉬움
- 감전: 얇아서 건너뛸 수 있음 → 수평 이동 요구
- 카메라: 방향성이 있음 → 우회 가능

#### 수비 턴 함정 배치

| 함정 | 비용 | 효과 | 재사용 쿨 | 전략성 |
|------|------|------|----------|--------|
| 레이저 | 2 | 탐지+피해 | 0.7초 | 중간 |
| 감전 바닥 | 2 | 지연 1초 | 1.4초 | 낮음 |
| 카메라 | 1 | 탐지만 | 1.2초 | 낮음 |
| 방화벽 | 3 | 지연+피해 | 1.8초 | 높음 |

**밸런싱 원칙:**
1. **비용과 효과 비례**: 비용 높음 = 강한 효과
2. **쿨타임 필수**: 같은 함정에 계속 걸리지 않도록
3. **조합 가능성**: 여러 함정을 같이 써야 효과 증가

**예시 (수비 턴 배치 전략):**
```
예산: 6

전략 1 (안전):
- 레이저 3개 → 탐지 3회 (목표: 2회 탐지)
- 감전 추가 불가능

전략 2 (지연):
- 방화벽 1개 + 감전 2개 → 지연 3초 (목표: 3초 지연)
- 탐지 없음 (목표: 탐지 2회 필요하면 실패)

전략 3 (균형):
- 레이저 2개 + 감전 1개 + 카메라 1개
- 탐지 2회 + 지연 1초
```

## ⏱️ 스테이지별 시간 설계

### 스테이지 시간 (팀원 D가 설정)

**원칙:**
- 공격 턴은 넉넉함 (실패 주요원인: 함정, 시간 아님)
- 초반은 길고, 후반으로 갈수록 짧음
- 무한모드는 계속 짧아짐

| 스테이지 | 턴 | 제한시간 | 기본 함정 수 | 목표 완료 시간 | 난이도 |
|---------|-----|---------|----------|---------|--------|
| 1 | 공격 | 48초 | 3개 | 25초 | 🟢 아주쉬움 |
| 2 | 수비 | - | 예산4 | 30초 | 🟢 아주쉬움 |
| 3 | 공격 | 48초 | 3개 + 1개 | 30초 | 🟢 쉬움 |
| 4 | 수비 | - | 예산5 | 40초 | 🟡 중간 |
| 5 | 공격 | 42초 | 4개 + 1개 | 30초 | 🟡 중간 |
| 6 | 수비 | - | 예산5 | 40초 | 🟡 중간 |
| 7 | 공격 | 42초 | 5개 | 35초 | 🟠 어려움 |
| 8 | 수비 | - | 예산6 | 45초 | 🟠 어려움 |
| 9 | 공격 | 38초 | 5개 + 1개 | 30초 | 🔴 매우어려움 |
| 10 | 수비 | - | 예산7 | 50초 | 🔴 매우어려움 |
| 11 | 공격 | 38초 | 5개 + 1개 | 30초 | 🔴 매우어려움 |
| 12+ | 무한 | 40→20초 | 증가 | - | 🔴 점진증가 |

**난이도 분포:**
```
1-2: 튜토리얼 (배우는 구간)
3-6: 기본 (기계 배우는 구간)
7-10: 고급 (실력 필요)
11: 클라이맥스 (모든 능력 필요)
12+: 엔드게임 (생존)
```

## 💰 보상 시스템 설계

### 공격 턴 성공 보상 (팀원 D가 관리)

**원칙:**
- 공격 성공 → 다음 수비 턴을 쉽게 함
- 수비를 너무 쉽게 하지 않기 (밸런스 유지)

| 보상 | 효과 | 다음 수비 턴에 미치는 영향 | 선택률 추정 |
|------|------|--------------------------|----------|
| 함정 예산 +2 | 함정 2개 추가 배치 | 크다 (예산 50% 증가) | 높음 |
| 카메라 비용 -1 | 카메라 무료 | 중간 (유연성 증가) | 낮음 |
| 레이저 강화 | 레이저 높이 +18px | 낮음 (탐지만) | 중간 |
| 방화벽 강화 | 지연시간 +0.5초 | 크다 (지연 전략 강화) | 높음 |

**추천 선택:**
```
초보자 1회차: 함정 예산 +2 (가장 이해하기 쉬움)
실력자 게임: 방화벽 강화 (고급 전략)
```

### 수비 턴 성공 보상

**원칙:**
- 수비 성공 → 다음 공격 턴을 쉽게 함
- 공격을 너무 강하게 하지 않기 (밸런스 유지)

| 보상 | 효과 | 다음 공격 턴에 미치는 영향 | 선택률 추정 |
|------|------|--------------------------|----------|
| 최대 에너지 +20 | 에너지 20% 증가 | 크다 (선택지 증가) | 높음 |
| 대시 쿨타임 -0.12초 | 대시 더 자주 사용 | 중간 (이동성 증가) | 중간 |
| 방어막 효율 증가 | 방어막 드레인 -4 | 낮음 (수비력) | 낮음 |
| 보호막 1회 | 함정 피해 1회 무시 | 중간 (보험) | 중간 |

**추천 선택:**
```
초보자: 최대 에너지 +20 (가장 실질적)
실력자: 대시 쿨타임 감소 (고급 이동)
```

## 🎯 보상 누적에 따른 난이도 조정

### 문제 상황: 보상이 계속 쉬워지는 현상

```
정상 플레이:       보상 받을 때마다:
1스테이지: 어렵음   → 매우쉬움
2스테이지: 쉬움     → 매우쉬움
3스테이지: 중간     → 쉬움
4스테이지: 중간     → 쉬움
...
```

### 해결책: 보상 한계 설정

```javascript
// MVP 제한
const maxRewardStacks = {
  defenseBudgetBonus: 3,      // 최대 +6 예산
  maxEnergy: 3,                // 최대 +60 에너지
  laserBoost: 2,               // 최대 +36px 레이저
  freeHit: 2,                  // 최대 2회 무시
};
```

**결과:**
- 보상이 무한정 누적되지 않음
- 약 7-8스테이지 이후 보상이 한계에 도달
- 플레이어가 다른 전략 필요하게 됨

## 🔢 정확한 수치 기준표

### 체력 시스템

| 상황 | 현재값 | 근거 |
|------|-------|------|
| 최대 체력 | 3 | 2회 실수 가능 (3회 목 죽음) |
| 레이저 피해 | -1 | 체력의 33% |
| 방화벽 피해 | -1 | 체력의 33% |
| 방어막으로 무시 | 제한없음 | 에너지가 주 제약 |

**예시:**
```
공격 턴에서 함정 3개 만남:
- 레이저 1개 피함
- 감전 2개: 에너지로 무시
- 레이저 1개에 맞음 → 체력 -1 (2/3)
- 결과: 클리어 가능
```

### 에너지 시스템

| 행동 | 소모 | 효율 | 사용 횟수/스테이지 |
|------|------|------|-----------------|
| 대시 | 18 | 높음 | 3-5회 |
| 방어막 (1초) | 24 | 중간 | 2-4회 |
| 방어막 (3초) | 72 | 낮음 | 0-1회 |

**에너지 회복 속도:**
```
30초 스테이지 × 10/s = 300 에너지 최대
- 대시 5회 = 90 (30%)
- 방어막 2회 (2초) = 48 (16%)
- 남은 여유 = 162 (54%) ← 여유 있음
```

**실력자 플레이어 기준:**

- 9-10스테이지: 20회 안에 클리어

## 🎮 조정 체크리스트 (2주차)

### 8일차: 1차 밸런싱

- [ ] 1-3스테이지가 너무 쉬운가? → 함정 수 증가
- [ ] 4-6스테이지가 너무 어려운가? → 보상 강화
- [ ] 보상이 선택지로 보이는가? → 또는 명확한 메타가 있는가?

- [ ] 9-11스테이지가 클리어 불가능한가? → 보상 조정
- [ ] 무한모드가 너무 빨리 어려워지는가? → 난이도 곡선 완화
- [ ] 보상 중 선택되지 않는 것이 있는가? → 효과 강화

## 💡 밸런싱 원칙 요약

1. **공격과 수비의 균형**
   - 공격 턴 보상 강함 → 수비 턴 더 어려워야 함
   - 수비 턴 성공 → 공격 턴 쉬워지면 다음 수비 턴 더 어려워야 함

2. **난이도 곡선**
   - 선형이 아니라 지수함수로 증가
   - 초반 넉넉, 후반 극단적

3. **보상 다양성**
   - 선택지가 의미 있어야 함
   - 최적 선택이 항상 같으면 안 됨

4. **반복 플레이성**
   - 매 게임이 다르게 느껴져야 함
   - 보상과 맵이 변하기 때문에 다른 전략 필요
