## 🎮 게임 룰 요약

### 공격 턴 (홀수 스테이지: 1, 3, 5, 7, 9, 11)

**당신:** 해커  
**목표:** 데이터 코어까지 도달해서 탈취  

**조작:**
- `A` / `D`: 이동
- `Space`: 점프
- `Shift`: 대시 (에너지 소모)
- `E`: 방어막 (에너지 소모)

**보상:** 공격 성공 시 보상 3개 중 1개 선택 → 다음 수비 턴에 적용

### 수비 턴 (짝수 스테이지: 2, 4, 6, 8, 10)

**당신:** AI 방어 시스템  
**목표:** 이전 해커의 움직임을 분석해서 함정 배치로 저지  

**조작:**
1. 함정 선택 (레이저, 감전, 카메라, 방화벽)
2. 맵의 슬롯 클릭으로 배치
3. "리플레이 시작" 클릭
4. 함정에 걸리는지 확인

**보상:** 수비 성공 시 보상 3개 중 1개 선택 → 다음 공격 턴에 적용

## 🧪 테스트 방법

### 일일 테스트 (15분)

모든 기능이 정상 작동하는지 확인:
1. 1스테이지 플레이 가능한가?
2. 2스테이지 함정 배치 가능한가?
3. 리플레이가 경로를 따르는가?
4. UI 모두 표시되는가?

### 주간 통합 테스트 (금요일, 30분)

- 1-11스테이지까지 통과 가능한가?
- 보상이 다음 턴에 적용되는가?
- 무한모드가 진행되는가?
- 버그나 튕김이 없는가?

## ⚠️ 주의사항

### 절대 해서는 안 될 것
1. ❌ game 객체 구조를 임의로 변경
2. ❌ 다른 팀원의 담당 함수 수정
3. ❌ 테스트 없이 커밋
4. ❌ 문제를 1시간 이상 혼자 해결하려고 시도

### 반드시 해야 할 것
1. ✅ TEAM_API_SPEC.md 지키기
